## 주요 기능
>#### 일반사용자
>사용자의 위치에 따라 근처의 장비 대여 업체들이 지도에 보인다
>업체를 선택하면 업체의 정보가 화면에 나타난다
>#### 업주
>회원가입하여 업체를 등록할 수 있다
>업체의 사진들을 등록할 수 있다 (대표사진 설정 가능)
>장비 목록을 선택하여 일반 사용자에게 보여줄 수 있다
